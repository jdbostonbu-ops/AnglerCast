User Stories — AnglerCast

Format: "As a [who], I want to [what] so that [why]."

#   User Story                                                                                              Priority

1   As a new fisherman, I want to create an account and verify my email with a code, so that my account     Must Do
    is secured and mine before I start saving spots.

2   As a returning fisherman, I want to log in with my email and password, so that I can get back to my      Must Do
    saved fishing spots.

3   As a logged-in fisherman, I want to save, view, edit, and delete my fishing spots, so that I can keep    Must Do
    track of the places I care about (with a confirmation before a delete so I don't lose one by accident).

4   As a fisherman, I want the app to recommend real spots — fish, location, and month — so that the app     Nice to Have
    is useful even when my own coordinates come up empty.

5   As a fisherman, I want to enter my location and get a conditions-aware travel time to a spot, so that     Stretch
    I can plan my trip around the weather and distance.
